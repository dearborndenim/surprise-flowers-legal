# Surprise Flowers — Legal

Public legal pages for [Surprise Flowers Co.](https://surpriseflowers.co).

- **`/sms-opt-in.html`** — SMS program terms / proof-of-opt-in URL for Twilio A2P 10DLC registration.

The same content is mirrored inside the production messaging service at `/legal/sms-opt-in` once that's deployed. This standalone repo exists so the public URL can be live before the messaging service has its env vars filled in.
